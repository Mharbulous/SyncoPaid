# Menu Bar

> **Last Updated:** 2025-12-25
> **Parent:** [Navigation Index](2025-12-25-Navigation-Index.md)

---

## Overview

SyncoPaid uses a traditional Windows menu bar to provide access to all application features. The menu bar appears at the top of the main window, below the title bar.

---

## Menu Bar Structure

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  File    Edit    View    Tools    Help                                      │
└─────────────────────────────────────────────────────────────────────────────┘
```

---

## File Menu

```
┌─────────────────────────────────┐
│  File                           │
├─────────────────────────────────┤
│  New Client          Ctrl+Shift+N│
│  New Matter          Ctrl+N      │
│  ─────────────────────────────  │
│  Import                    ▶    │──┬─────────────────────────┐
│  Export                    ▶    │  │  Import Clients (CSV)   │
│  ─────────────────────────────  │  │  Import Matters (CSV)   │
│  Settings              Ctrl+,   │  │  Import Client-Matters  │
│  ─────────────────────────────  │  └─────────────────────────┘
│  Minimize to Tray               │
│  Exit                  Alt+F4   │
└─────────────────────────────────┘
         │
         └──▶ Export submenu
              ┌─────────────────────────┐
              │  Export Activities...   │
              │  Export Report...       │
              │  Quick Export (JSON)    │
              │  Quick Export (CSV)     │
              └─────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| New Client | Ctrl+Shift+N | Opens New Client dialog |
| New Matter | Ctrl+N | Opens New Matter dialog |
| Import → Import Clients (CSV) | — | Opens file picker for client CSV import |
| Import → Import Matters (CSV) | — | Opens file picker for matter CSV import |
| Import → Import Client-Matters | — | Opens file picker for combined import |
| Export → Export Activities... | — | Opens [Export View](2025-12-25-Export-View.md) |
| Export → Export Report... | — | Opens report export dialog |
| Export → Quick Export (JSON) | — | Exports current view to JSON |
| Export → Quick Export (CSV) | — | Exports current view to CSV |
| Settings | Ctrl+, | Opens [Settings Modal](2025-12-25-Settings-Modal.md) |
| Minimize to Tray | — | Hides window, keeps running in tray |
| Exit | Alt+F4 | Closes application completely |

## Edit Menu

```
┌─────────────────────────────────┐
│  Edit                           │
├─────────────────────────────────┤
│  Undo                    Ctrl+Z │
│  Redo                    Ctrl+Y │
│  ─────────────────────────────  │
│  Select All              Ctrl+A │
│  ─────────────────────────────  │
│  Assign to Matter...     Ctrl+M │
│  Edit Narrative...       Ctrl+E │
│  Split Activity...              │
│  Merge Activities...            │
│  ─────────────────────────────  │
│  Delete                  Del    │
└─────────────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Undo | Ctrl+Z | Undo last action |
| Redo | Ctrl+Y | Redo last undone action |
| Select All | Ctrl+A | Select all items in current view |
| Assign to Matter... | Ctrl+M | Opens Matter Picker for selected activities |
| Edit Narrative... | Ctrl+E | Edit billing narrative for selected activity |
| Split Activity... | — | Opens Split Activity dialog |
| Merge Activities... | — | Merge selected activities into one |
| Delete | Del | Delete selected items (with confirmation) |

│  ─────────────────────────────  │
│  Filter                    ▶    │──┬─────────────────────────┐
│  ─────────────────────────────  │  │  By Date Range...       │
│  Refresh                     F5 │  │  By Client...           │

                                     │  By Application...      │
                                     │  ─────────────────────  │
                                     │  Clear Filters          │
                                     └─────────────────────────┘

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Timeline | F2 | Switch to [Timeline View](2025-12-25-Timeline-View.md) |
| Activities | F3 | Switch to [Activities View](2025-12-25-Activities-View.md) |
| Matters | F4 | Switch to [Matters View](2025-12-25-Matters-View.md) |
| Clients | F5 | Switch to [Clients View](2025-12-25-Clients-View.md) |
| Reports | F6 | Switch to [Reports View](2025-12-25-Reports-View.md) |
| Export | F7 | Switch to [Export View](2025-12-25-Export-View.md) |
| Sidebar | Ctrl+B | Toggle sidebar visibility |
| Status Bar | — | Toggle status bar visibility |
| Zoom In | Ctrl++ | Increase timeline zoom level |
| Zoom Out | Ctrl+- | Decrease timeline zoom level |
| Reset Zoom | Ctrl+0 | Reset zoom to default |
| Filter → By Date Range... | — | Opens date range picker |
| Filter → By Client... | — | Opens client filter dropdown |
| Filter → By Matter... | — | Opens matter filter dropdown |
| Filter → By Application... | — | Opens application filter dropdown |
| Filter → Clear Filters | — | Remove all active filters |
| Refresh | F5 | Refresh current view data |
