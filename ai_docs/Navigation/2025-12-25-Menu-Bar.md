# Menu Bar

> **Last Updated:** 2025-12-26
> **Parent:** [Navigation Index](2025-12-25-Navigation-Index.md)

---

## Overview

SyncoPaid uses a traditional Windows menu bar to provide access to all application features. The menu bar appears at the top of the main window, below the title bar.

---

## Menu Bar Structure

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  File    Edit    View    Tools    Help                                      │
└─────────────────────────────────────────────────────────────────────────────┘
```

---

## File Menu

```
┌─────────────────────────────────────┐
│  File                               │
├─────────────────────────────────────┤
│  Import Folders...         Ctrl+I   │
│  ─────────────────────────────────  │
│  Review Billing...         Ctrl+B   │
│  Export                        ▶    │──┬─────────────────────────┐
│  ─────────────────────────────────  │  │  Export Activities...   │
│  Settings                  Ctrl+,   │  │  Quick Export (CSV)     │
│  ─────────────────────────────────  │  └─────────────────────────┘
│  Minimize to Tray                   │
│  Exit                      Alt+F4   │
└─────────────────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Import Folders... | Ctrl+I | Opens folder picker to import client/matter folder structure |
| Review Billing... | Ctrl+B | Opens [Billing Review Dialog](2025-12-25-Billing-Review-Dialog.md) |
| Export → Export Activities... | — | Opens [Export Dialog](../Mockups/2025-12-25-Export-View.md) |
| Export → Quick Export (CSV) | — | Exports current view to CSV |
| Settings | Ctrl+, | Opens [Settings Modal](2025-12-25-Settings-Modal.md) |
| Minimize to Tray | — | Hides window, keeps running in tray |
| Exit | Alt+F4 | Closes application completely |

**Note:** We import folder structures from the user's file system — we don't create or manage clients/matters. See [Terminology](../../../CLAUDE.md#terminology).

## Edit Menu

```
┌─────────────────────────────────┐
│  Edit                           │
├─────────────────────────────────┤
│  Undo                    Ctrl+Z │
│  Redo                    Ctrl+Y │
│  ─────────────────────────────  │
│  Select All              Ctrl+A │
│  ─────────────────────────────  │
│  Change Matter...        Ctrl+M │
│  Edit Narrative...       Ctrl+E │
│  Split Activity...              │
│  Merge Activities...            │
│  ─────────────────────────────  │
│  Delete                  Del    │
└─────────────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Undo | Ctrl+Z | Undo last action |
| Redo | Ctrl+Y | Redo last undone action |
| Select All | Ctrl+A | Select all items in current view |
| Change Matter... | Ctrl+M | Correct AI's matter assignment for selected activities |
| Edit Narrative... | Ctrl+E | Correct AI-generated billing narrative |
| Split Activity... | — | Correct AI's activity boundaries |
| Merge Activities... | — | Correct AI's activity boundaries by merging |
| Delete | Del | Delete selected items (with confirmation) |

**Note:** Change Matter, Edit Narrative, Split, and Merge are **correction actions** — used when AI got something wrong. The primary workflow is accepting AI suggestions in the [Review Activities View](../Mockups/2025-12-25-Activities-View.md).

## View Menu

```
┌─────────────────────────────────┐
│  View                           │
├─────────────────────────────────┤
│  Timeline                    F2 │
│  Review Activities           F3 │
│  Client Matters              F4 │
│  ─────────────────────────────  │
│  Status Bar                     │
│  ─────────────────────────────  │
│  Zoom In                 Ctrl++ │
│  Zoom Out                Ctrl+- │
│  Reset Zoom              Ctrl+0 │
│  ─────────────────────────────  │
│  Filter                    ▶    │──┬─────────────────────────┐
│  ─────────────────────────────  │  │  By Date Range...       │
│  Refresh                     F5 │  │  By Application...      │
└─────────────────────────────────┘  │  ─────────────────────  │
                                     │  Clear Filters          │
                                     └─────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Timeline | F2 | Switch to [Timeline View](2025-12-25-Timeline-View.md) |
| Review Activities | F3 | Switch to [Review Activities View](../Mockups/2025-12-25-Activities-View.md) |
| Client Matters | F4 | Switch to [Client Matters List](../Mockups/2025-12-25-Client-Matters-List.md) (imported folders) |
| Status Bar | — | Toggle status bar visibility |
| Zoom In | Ctrl++ | Increase timeline zoom level |
| Zoom Out | Ctrl+- | Decrease timeline zoom level |
| Reset Zoom | Ctrl+0 | Reset zoom to default |
| Filter → By Date Range... | — | Opens date range picker |
| Filter → By Application... | — | Opens application filter dropdown |
| Filter → Clear Filters | — | Remove all active filters |
| Refresh | F5 | Refresh current view data |

**Note:** Export and Reports are **actions** (dialogs), not views. Access Export via File menu. Billing triage is handled by File → Review Billing.

## Tools Menu

```
┌─────────────────────────────────┐
│  Tools                          │
├─────────────────────────────────┤
│  Start Tracking          Ctrl+T │
│  Pause Tracking          Ctrl+P │
│  ─────────────────────────────  │
│  Run AI Categorization   Ctrl+G │
│  ─────────────────────────────  │
│  Clean Up Screenshots...        │
│  ─────────────────────────────  │
│  Database                  ▶    │──┬─────────────────────────┐
│                                 │  │  View Statistics        │
└─────────────────────────────────┘  │  Compact Database       │
                                     │  Backup Database...     │
                                     │  Restore Database...    │
                                     └─────────────────────────┘
```

| Menu Item | Shortcut | Action |
|-----------|----------|--------|
| Start Tracking | Ctrl+T | Start window tracking |
| Pause Tracking | Ctrl+P | Pause window tracking |
| Run AI Categorization | Ctrl+G | Queue uncategorized activities for AI processing |
| Clean Up Screenshots... | — | Opens screenshot cleanup dialog |
| Database → View Statistics | — | Show database size and record counts |
| Database → Compact Database | — | Optimize database file size |
| Database → Backup Database... | — | Export database backup |
| Database → Restore Database... | — | Restore from backup file |

**Note:** Screenshots are used by AI for context during categorization — not for standalone browsing. See [non-goals](../../../.claude/data/goals/non-goals.md#standalone-screenshot-galleryviewer).
